# Rixly Landing Page - PRD

## Original Problem Statement
Create a professional landing page for Rixly - AI-powered lead generation SaaS with:
- Simple beautiful motions
- Walkthrough video section
- Realistic testimonials with real faces
- Minimal text, clean design
- Dark/Light theme toggle

## User Personas
- **Primary**: Startup founders, sales managers seeking lead generation solutions
- **Secondary**: Marketing directors at SMBs looking for AI-powered tools

## Core Requirements
- Dark theme default with light theme toggle
- Fresh unique color palette (Electric Coral #FF6B4A on Digital Obsidian #02040A)
- CTAs: Get Started, Book a Demo, Pricing
- Responsive mobile-first design
